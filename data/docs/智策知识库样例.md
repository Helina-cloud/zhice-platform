# 智策（ZhiCe）平台说明

智策是企业智能知识助手平台，用于统一加载内部文档、构建检索索引，并通过大模型生成可追溯的回答。

## 基础版能力

基础版提供 RAG 引擎：从 `data/docs` 加载文档，使用本地 FAISS 索引（`data/vector_db`）存储嵌入；检索阶段结合稠密向量检索与 BM25 词面匹配；问答阶段使用 LangChain LCEL 将检索、提示词与聊天模型串联。生产环境也可替换为 Chroma 等远程向量库。

## 数据与安全

生产环境需在 `.env` 中配置 API 密钥，并限制文档目录的访问权限。回答应仅依据检索到的上下文，不得臆造未在知识库中出现的政策或数值。
